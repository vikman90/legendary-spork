# Wazuh Agent Documentation Demo

This is a temporary PoC: https://vikman90.github.io/legendary-spork

## Setup

### Requirements

- mkdocs

Install with PIP:

```shell
pip3 install mkdocs
```

### Build documentation

```shell
mkdocs build
```

### Serve documentation

```shell
mkdocs serve
```

This will make documentation available at http://127.0.0.1:8000.
